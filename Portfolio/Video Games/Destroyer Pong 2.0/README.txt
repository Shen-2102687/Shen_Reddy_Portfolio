This game is inspired by the retro game "Pong".

The player must try to destroy the falling meteors to score points, while keeping the
bouncing ball in the air.

Points are scored for each meteor destroyed. Player health is lost everytime the ball falls
off the screen. The game ends when all player health is lost.

Download/Install Instructions:

-Please download and extract the Build.zip file
-Run the pongaasignment.exe file to play the game.

Play Instructions:

-Move the paddle left and right with the respective arrow keys
-Shoot at the meteors using spacebar
-Prevent the ball from falling by placing the paddle underneath it before it falls below
the screen to hit it back up in the air.

Exit the game using ALT+f4
